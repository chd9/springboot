# springboot-layui

## 项目介绍
基于springboot+layui简单后台管理系统，好看好用，易于二次开发。
项目功能包括用户管理、权限管理、菜单管理以及系统设置。
## 运行环境
1. Linux
2. JDK1.8 及以上版本
3. mysql5.1.45

## 安装教程

1. 创建数据库manage，将src/main/resources目录下manage.sql导入
2. 修改application.yml中数据库连接配置
3. ManageApplication为项目启动入口
