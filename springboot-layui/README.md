#### 运行环境
1. Linux
2. JDK1.8 及以上版本
3. mysql5.1.45

#### 安装教程

1. 创建数据库manage，将src/main/resources目录下manage.sql导入
2. 修改application.yml中数据库连接配置
3. ManageApplication为项目启动入口


后台默认用户名密码：admin/admin
